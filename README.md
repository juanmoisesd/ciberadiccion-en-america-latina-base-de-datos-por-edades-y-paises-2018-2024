## Overview
Base de datos sobre ciberadicción (adicción a internet, redes sociales y videojuegos) en América Latina, con estadísticas organizadas por grupos de edad (niños, adolescentes, adultos jóvenes, adultos, adultos mayores) y por países (México, Brasil, Argentina, Colombia, Chile, Perú, Ecuador, Bolivia, Paraguay, Uruguay, Venezuela, Costa Rica, Guatemala, Honduras, Panamá, República Dominicana).
